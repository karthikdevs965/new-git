# 🔐 Login & Authentication System

A secure and user-friendly **Authentication System** that allows users to **Register, Login, and Access Protected Pages**.  
This project demonstrates core concepts of **web security, form validation, password hashing, token-based authentication (JWT), and route protection.**

---

## ✅ Features

✔ User Registration (Signup)  
✔ Secure Login (Authentication)  
✔ Password hashing (bcrypt / crypto)  
✔ JWT-based authentication  
✔ Protected routes (only logged-in users can access)  
✔ Logout functionality  
✔ Error handling & form validation  
✔ Responsive UI  
✔ LocalStorage / Cookies / Sessions (based on implementation)

---

## 🛠️ Tech Stack

- HTML, CSS, JavaScript, Python
---


## 🔐 Authentication Flow

1️⃣ User registers → data saved with **hashed password**  
2️⃣ User logs in → server validates credentials  
3️⃣ Server returns **JWT token**  
4️⃣ Token is stored in **LocalStorage / Cookies**  
5️⃣ Token is sent in headers for protected APIs  
6️⃣ If token is valid → access granted  
7️⃣ Logout → token removed, session ends
